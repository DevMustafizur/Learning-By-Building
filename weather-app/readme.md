# 🌤️ Weather App (HTML, CSS, JavaScript)

A simple weather application built using **HTML, CSS, and JavaScript**.  
This project shows real-time weather information like temperature, humidity, wind speed, and rain chance.

---

## 🚀 Features

- 🌡️ Current temperature display
- 🌍 City / location search
- 💧 Humidity information
- 🌬️ Wind speed display
- 🌧️ Rain probability (optional API feature)
- 🎨 Clean and responsive UI design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome (icons)
- OpenWeather API

## 🔍 How It Works

1. User enters a city name
2. JavaScript fetches weather data from API
3. UI updates dynamically with weather info
