
# Calculator App

A modern calculator application created with pure HTML, CSS, and JavaScript.

## 🚀 Features

- Addition, subtraction, multiplication, and division
- Responsive UI
- User-friendly design
- Fast and lightweight

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## 📂 Files

- `index.html` → Structure
- `style.css` → Styling
- `script.js` → Functionality
